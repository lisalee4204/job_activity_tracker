## Next Steps: Build Your APK

### Quick Version (3 Commands)

```bash
# 1. Build web app
cd frontend
npm run build

# 2. Sync to Android
npx cap sync

# 3. Open in Android Studio
npx cap open android
```

Then in Android Studio:
- **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
- Wait 2-5 minutes
- Your APK will be ready!
