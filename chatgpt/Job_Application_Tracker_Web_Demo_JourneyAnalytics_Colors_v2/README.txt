# Job Application Tracker (Demo)

## Open-and-go
- Double-click `index.html` to open in a browser.
- Demo data is preloaded.
- Edit any cell inline. Your changes auto-save in your browser (localStorage).

## Portable data
- Use **Export JSON** to download your current dataset.
- Use **Import JSON** to load it on another computer.

## Optional: host it
Deploy this folder to Netlify / Vercel / GitHub Pages for a sharable link.
