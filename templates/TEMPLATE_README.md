# Job Search Activity Tracker - Excel/Google Sheets Template

## Overview

This template allows you to track your job search activities in Excel or Google Sheets before importing them into the application, or as a backup/offline tracking method.

## Template Structure

The template includes the following columns:

### Required Fields
- **Date**: Date of the activity (YYYY-MM-DD format)
- **Company Name**: Name of the company
- **Job Title**: Title of the position
- **Activity Type**: Type of activity (see options below)

### Optional Fields
- **Status**: Current status of the application (see options below)
- **Job Description URL**: Link to the job posting
- **Contact Person**: Name of the person you contacted
- **Contact Method**: How you contacted them (Email, Phone, LinkedIn, etc.)
- **Notes**: Additional notes about the activity
- **Source**: Where you found the job posting (see options below)
- **Location**: Job location (city, state, or Remote)
- **Salary Range**: Expected or offered salary range

## Field Options

### Activity Types
- `application` - Job application submitted
- `interview` - Interview scheduled or completed
- `networking` - Networking event or contact
- `job_fair` - Job fair attendance
- `resume_submission` - Resume submitted
- `phone_call` - Phone call with recruiter/employer
- `email_inquiry` - Email inquiry sent
- `recruiter_contact` - Contacted by recruiter
- `other` - Other activity type

### Status Options
- `application` - Application submitted
- `assessment` - Assessment/test completed
- `hr_screen` - HR screening call
- `hiring_manager` - Interview with hiring manager
- `final_round` - Final round interview
- `offer` - Job offer received
- `rejected` - Application rejected

### Source Options
- `linkedin` - Found on LinkedIn
- `indeed` - Found on Indeed
- `company_website` - Company's career page
- `glassdoor` - Found on Glassdoor
- `monster` - Found on Monster
- `other` - Other source
- `gmail_import` - Imported from Gmail (auto-filled)

## Usage Instructions

### Excel
1. Open the CSV file in Excel
2. Excel will automatically format it as a table
3. Fill in your job search activities
4. Save as CSV or Excel format (.xlsx)
5. Import into the application when ready

### Google Sheets
1. Upload the CSV file to Google Drive
2. Right-click and select "Open with Google Sheets"
3. Fill in your job search activities
4. Download as CSV when ready to import
5. Or use Google Sheets as your primary tracking tool

## Importing into the Application

1. Export your data as CSV from Excel/Google Sheets
2. Ensure the column headers match exactly
3. Use the import feature in the application (if available)
4. Or manually enter activities using the "Add Activity" form

## Tips for Effective Tracking

1. **Be Consistent**: Log activities as soon as they happen
2. **Include Details**: The more information you log, the better insights you'll get
3. **Update Status**: Regularly update the status of your applications
4. **Use Notes**: Add context about conversations, next steps, or important details
5. **Track Sources**: Note where you found each opportunity to identify successful channels

## Troubleshooting

### Date Format Issues
- Use YYYY-MM-DD format (e.g., 2024-01-15)
- Excel may auto-format dates - ensure they're saved correctly

### Import Errors
- Check that all required fields are filled
- Verify activity types and statuses match the allowed values
- Ensure dates are in the correct format
- Remove any special characters that might cause issues

### Missing Data
- Required fields must have values
- Optional fields can be left empty
- Use consistent formatting for better data quality
